```python
update_claude_config(file_spec: str, server_name: str) -> bool
```


在 Claude 配置中添加或更新 FastMCP 服务器。

**参数：**
- `file_spec`: 服务器文件的路径，可选择性地带有 \:object 后缀
- `server_name`: 在 Claude 配置中的服务器名称
- `with_editable`: 可选的目录，以可编辑模式安装
- `with_packages`: 可选的附加包列表，用于安装
- `env_vars`: 可选的环境变量字典。这些变量会与现有变量合并，新值优先。

**抛出异常：**
- `RuntimeError`: 如果未找到 Claude Desktop 的配置目录，表明 Claude Desktop 可能未安装或未正确设置。

```python
make_middleware_wrapper(middleware: Middleware, call_next: CallNext[T, R]) -> CallNext[T, R]
```


创建一个将单个中间件应用于上下文的包装器。
闭包中包含中间件和 call_next 函数，因此可以将其
传递给其他期望 call_next 函数的函数。

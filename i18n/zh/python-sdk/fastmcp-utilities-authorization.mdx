# `fastmcp.utilities.authorization`


FastMCP 组件的授权检查。

授权检查是接收 ``AuthContext`` 的可调用对象，返回 True 表示允许访问，返回 False 表示拒绝访问。它们也可以抛出 ``AuthorizationError`` 并附带自定义消息来拒绝访问；其他异常会被屏蔽并视为拒绝。

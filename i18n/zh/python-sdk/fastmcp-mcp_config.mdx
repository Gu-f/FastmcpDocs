# `fastmcp.mcp_config`


规范的 MCP 配置格式。

此模块定义了模型上下文协议 (MCP) 服务器的标准配置格式。
它提供了一个客户端无关的可扩展格式，可以在所有 MCP 实现中使用。

该配置格式支持 stdio 和远程 (HTTP/SSE) 传输，为服务器元数据、
身份验证和执行参数提供全面的字段定义。

配置示例：
```json
{
    "mcpServers": {
        "my-server": {
            "command": "npx",
            "args": ["-y", "@my/mcp-server"],
            "env": {"API_KEY": "secret"},
            "timeout": 30000,
            "description": "My MCP server"
        }
    }
}
```

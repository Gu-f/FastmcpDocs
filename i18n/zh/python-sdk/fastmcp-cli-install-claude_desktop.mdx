### `install_claude_desktop` <sup><a href="https://github.com/jlowin/fastmcp/blob/main/src/fastmcp/cli/install/claude_desktop.py#L37" target="_blank"><Icon icon="github" style="width: 14px; height: 14px;" /></a></sup>

```python
install_claude_desktop(file: Path, server_object: str | None, name: str) -> bool
```


在 Claude Desktop 中安装 FastMCP 服务器。

**参数：**
- `file`: 服务器文件的路径
- `server_object`: 可选的服务器对象名称（用于 \:object 后缀）
- `name`: 在 Claude 配置中的服务器名称
- `with_editable`: 可选的目录，以可编辑模式安装
- `with_packages`: 可选的附加包列表，用于安装
- `env_vars`: 可选的环境变量字典
- `python_version`: 可选的要使用的 Python 版本
- `with_requirements`: 可选的需求文件，用于安装依赖
- `project`: 可选的项目目录，在其中运行

**返回值：**
- 如果安装成功则返回 True，否则返回 False

MCP 服务端可以在长时间运行的操作期间报告进度。客户端可以通过进度处理器接收这些更新。

## 进度处理器

创建客户端时设置进度处理器：

```python
from fastmcp import Client

async def my_progress_handler(
    progress: float, 
    total: float | None, 
    message: str | None
) -> None:
    if total is not None:
        percentage = (progress / total) * 100
        print(f"进度：{percentage:.1f}% - {message or ''}")
    else:
        print(f"进度：{progress} - {message or ''}")

client = Client(
    "my_mcp_server.py",
    progress_handler=my_progress_handler
)
```

### 处理器参数

进度处理器接收三个参数：


<Card icon="code" title="进度处理器参数">
<ResponseField name="progress" type="float">
  当前进度值
</ResponseField>

<ResponseField name="total" type="float | None">
  预期总值（可能为 None）
</ResponseField>

<ResponseField name="message" type="str | None">
  可选状态消息（可能为 None）
</ResponseField>
</Card>


## 按调用设置进度处理器

为特定工具调用覆盖进度处理器：

```python
async with client:
    # 为此调用覆盖为特定进度处理器
    result = await client.call_tool(
        "long_running_task", 
        {"param": "value"}, 
        progress_handler=my_progress_handler
    )
```

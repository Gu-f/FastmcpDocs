**FastMCP 是构建 MCP 应用的标准框架。** [Model Context Protocol](https://modelcontextprotocol.io/)（MCP）提供了一种标准化方式，将 LLM 连接到工具和数据；FastMCP 则用清晰、Pythonic 的代码让它达到生产就绪：

```python {1}
from fastmcp import FastMCP

mcp = FastMCP("Demo 🚀")

@mcp.tool
def add(a: int, b: int) -> int:
    """两个数字相加"""
    return a + b

if __name__ == "__main__":
    mcp.run()
```

## 什么是 MCP？

Model Context Protocol 让你能够构建服务端，以安全、标准化的方式向 LLM 应用暴露数据和功能。它常被描述为“AI 的 USB-C 接口”，为 LLM 连接可用资源提供统一方式。你也可以把它理解为一种 API，但它是专为 LLM 交互设计的。MCP 服务端可以：

- 通过 `Resources` 暴露数据（可以把它们近似理解为 GET 端点；用于将信息加载到 LLM 的上下文中）
- 通过 `Tools` 提供功能（有点像 POST 端点；用于执行代码或产生其他副作用）
- 通过 `Prompts` 定义交互模式（用于 LLM 交互的可复用模板）
- 以及更多！

FastMCP 提供高级、Pythonic 的接口，用于构建、管理并与这些服务端交互。

## 为什么选择 FastMCP？

FastMCP 会处理所有复杂的协议细节，让你专注于构建。大多数情况下，你只需要装饰一个 Python 函数，其余部分交给 FastMCP。

🚀 **快速**：高级接口意味着更少代码和更快开发

🍀 **简单**：用最少样板代码构建 MCP 服务端

🐍 **Pythonic**：对 Python 开发者来说感觉自然

🔍 **完整**：生产所需的一切，包括企业认证（Google、GitHub、Azure、Auth0、WorkOS）、部署工具、测试框架、客户端库等

FastMCP 提供从想法到生产的最短路径。你可以部署到本地、通过 [Prefect Horizon](https://horizon.prefect.io?utm_source=gofastmcp&utm_medium=docs) 部署到云端（个人项目免费），或部署到自己的基础设施。

## LLM 友好文档

FastMCP 文档提供多种 LLM 友好格式：

### MCP 服务端

FastMCP 文档可以通过 MCP 访问！服务端 URL 是 `https://gofastmcp.com/mcp`。

事实上，你可以使用 FastMCP 搜索 FastMCP 文档：

```python
import asyncio
from fastmcp import Client

async def main():
    async with Client("https://gofastmcp.com/mcp") as client:
        result = await client.call_tool(
            name="search_fast_mcp",
            arguments={"query": "deploy a FastMCP server"}
        )
    print(result)

asyncio.run(main())
```

### 文本格式

文档也提供 [llms.txt format](https://llmstxt.org/)：
- [llms.txt](https://gofastmcp.com/llms.txt) - 列出所有文档页面的 sitemap
- [llms-full.txt](https://gofastmcp.com/llms-full.txt) - 单个文件中的完整文档（可能超过上下文窗口）

任何页面都可以通过在 URL 后追加 `.md` 作为 markdown 访问。例如，本页变为 `https://gofastmcp.com/getting-started/welcome.md`。

你也可以按键盘上的 “Cmd+C”（Windows 上为 “Ctrl+C”）将任意页面复制为 markdown。

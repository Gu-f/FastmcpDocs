**FastMCP 是构建 MCP 应用程序的标准框架。** [模型上下文协议](https://modelcontextprotocol.io/) (MCP) 提供了一种标准化的方式来连接 LLM 与工具和数据，而 FastMCP 通过干净、Pythonic 的代码使其投入生产使用：

```python {1}
from fastmcp import FastMCP

mcp = FastMCP("演示 🚀")

@mcp.tool
def add(a: int, b: int) -> int:
    """两数相加"""
    return a + b

if __name__ == "__main__":
    mcp.run()
```


## 超越基础 MCP

FastMCP 开创了 Python MCP 开发，FastMCP 1.0 在 2024 年被纳入[官方 MCP SDK](https://github.com/modelcontextprotocol/python-sdk)。

**这是 FastMCP 2.0** — 积极维护的版本，远超基础协议实现。虽然 SDK 提供核心功能，但 FastMCP 2.0 提供生产所需的一切：高级 MCP 模式（服务器组合、代理、OpenAPI/FastAPI 生成、工具转换）、企业级认证（Google、GitHub、Azure、Auth0、WorkOS 等）、部署工具、测试框架和完整的客户端库。

准备开始构建？从我们的[安装指南](/zh/getting-started/installation)开始，或直接跳转到[快速开始](/zh/getting-started/quickstart)。


## 什么是 MCP？

模型上下文协议允许您构建服务器，以安全、标准化的方式向 LLM 应用程序公开数据和功能。它经常被描述为"AI 的 USB-C 接口"，提供了连接 LLM 与其可用资源的统一方式。您可以将其视为一个 API，但专门为 LLM 交互而设计。MCP 服务器可以：

- 通过 `Resources` 公开数据（可以将其视为类似 GET 端点；用于将信息加载到 LLM 的上下文中）
- 通过 `Tools` 提供功能（类似 POST 端点；用于执行代码或产生副作用）
- 通过 `Prompts` 定义交互模式（LLM 交互的可重用模板）
- 以及更多！

FastMCP 提供了一个高级、Pythonic 的接口来构建、管理和与这些服务器交互。

## 为什么选择 FastMCP？

FastMCP 处理所有复杂的协议细节，让您专注于构建。在大多数情况下，装饰一个 Python 函数就是您所需要的 — FastMCP 处理其余部分。

🚀 **快速**: 高级接口意味着更少的代码和更快的开发

🍀 **简单**: 以最少的样板代码构建 MCP 服务器

🐍 **Pythonic**: 对 Python 开发者来说感觉自然

🔍 **完整**: 生产所需的一切 — 企业级认证（Google、GitHub、Azure、Auth0、WorkOS）、部署工具、测试框架、客户端库等

FastMCP 提供从想法到生产的最短路径。可以本地部署、使用 [FastMCP Cloud](https://fastmcp.cloud)（个人服务器免费）部署到云端，或部署到您自己的基础设施。

FastMCP 由 [Prefect](https://www.prefect.io/) 用 💙 制作。



## LLM 友好的文档

FastMCP 文档提供多种 LLM 友好的格式：

### MCP 服务器

FastMCP 文档可通过 MCP 访问！服务器 URL 是 `https://gofastmcp.com/mcp`。

实际上，您可以使用 FastMCP 来搜索 FastMCP 文档：

```python
import asyncio
from fastmcp import Client

async def main():
    async with Client("https://gofastmcp.com/mcp") as client:
        result = await client.call_tool(
            name="SearchFastMcp",
            arguments={"query": "部署 FastMCP 服务器"}
        )
    print(result)

asyncio.run(main())
```

### 纯文本格式

文档还提供 [llms.txt 格式](https://llmstxt.org/)：
- [llms.txt](https://gofastmcp.com/llms.txt) - 列出所有文档页面的站点地图
- [llms-full.txt](https://gofastmcp.com/llms-full.txt) - 一个文件中的完整文档（可能超出上下文窗口）

任何页面都可以通过在 URL 后添加 `.md` 作为 markdown 格式访问。例如，此页面变为 `https://gofastmcp.com/getting-started/welcome.md`。

您还可以通过按键盘上的“Cmd+C”（或Windows上的“Ctrl+C”）将任何页面复制为markdown。
